# 회원가입 & 이메일 인증

Spring Boot 백엔드 + Next.js 프론트로 구현한 기본 회원가입/로그인 + 이메일 인증입니다.

## 주요 기능

### 1. 회원가입

- 이메일, 아이디, 닉네임, 비밀번호를 입력해 회원가입
- 비밀번호는 BCrypt로 해시 처리해서 DB에 저장
- 회원가입 성공 시 브라우저 팝업으로 안내 후 `/login` 페이지로 이동

### 2. 이메일 인증

- 이메일 입력 후 "인증" 버튼을 누르면 인증 코드 메일 전송 시도
- 0–9, A–Z로 이루어진 6자리 랜덤 코드를 생성해서 발송
- 사용자는 메일에서 코드를 복사해와 "인증 코드" 입력 + "확인" 버튼으로 검증
- 정상 인증 시:
  - "이메일 인증이 완료되었습니다." 문구 표시
  - 이메일/코드 입력칸이 비활성화됨
- 이메일 인증이 완료되어야만 최종 "가입하기" 버튼이 동작

### 3. 이미 가입된 이메일 처리

- 인증 버튼을 누를 때, 해당 이메일로 가입된 계정이 이미 있으면:
  - 인증 메일을 보내지 않고
  - “이미 가입된 이메일입니다. 로그인하러 가시겠습니까?” 팝업 표시
  - 확인을 누르면 `/login` 페이지로 이동

### 4. 백엔드 요약

- `POST /api/v1/member/email/send-code`  
  → 이메일로 인증 코드 전송 (이미 가입된 이메일이면 안내만 하고 메일 미전송)
- `POST /api/v1/member/email/check-code`  
  → 이메일 + 코드로 인증 여부 확인
- `POST /api/v1/member/signup`  
  → 이메일 인증된 경우에만 회원 생성
- `POST /api/v1/member/login`  
  → 아이디 / 비밀번호로 로그인 (비밀번호 해시 검증)

### 5. 프론트엔드 요약

- `/signup` 페이지
  - 이메일 인증 + 코드 확인 + 회원가입 한 번에 진행
  - 가입 성공 시 팝업 후 `/login`으로 리다이렉트
- 환경 변수: `NEXT_PUBLIC_API_BASE_URL`  
  → 백엔드 주소 설정 (예: `http://localhost:8080`)
